# CONTRIBUTING RULES

</div>

To participate in the development of the NeuroFix project for various users, you can submit an Issue using a specialized template. After this, you will need to wait a certain amount of time, and if you are accepted, you can become an official developer of the NeuroFix project.

[Issue Project (click)](https://github.com/AlbeePython/NeuroFix/issues)
