## 📋 Описание проекта

**NeuroFix** — это легковесная утилита на Python, созданная для восстановления беспрепятственного доступа к ИИ-сервисам и их API в регионах, где они официально заблокированы или недоступны.

Скрипт работает на уровне сетевых запросов, перенаправляя весь трафик вашего интернета на другой айпи во время захода на заблокированные AI сервисы в ином случаи скрипт никак себя и нигде не использует.

### ✨ Ключевые возможности:
* **Умный роутинг:** Проксируются только запросы к заблокированным ИИ (OpenAI, Anthropic, Gemini и др.).
* **Локальная производительность:** Нейросети будут работать от вашего интернета, скрипт только открывает доступ к заблокированым AI сервисам а сама работа / пинг и другое зависит уже от вашего интернет соеденения!
* **Поддержка API и Web:** Подходит как для работы в браузере, так и для интеграции в твои собственные Python-скрипты и ботов.
* **Простая настройка:** Запуск в несколько действий буз сложной заморочки.

## 🚀 Быстрый старт

### Требования
Убедись, что у тебя установлены:
* Python версии 3.8 или выше
* Установщик пакетов `pip`

### Установка (OpenSource эдитация)
- Убедитесь что у вас установлен `Python` с библиотекой `Tkinter`
- Откройте ПО из релизов
- Запускайте .py файл
- В открвшеемся окне выберите кнопку Начать
- Пробуйте заходить на AI сервисы ( gemini.google.com chatgpt.com claude.ai ) и другие

*Проект выполнен в соотвествии с лицензией Apache 2.0*
